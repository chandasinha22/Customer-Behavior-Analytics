# 📊 Customer Intelligence & Behavior Analytics

An end-to-end data analytics project focused on analyzing customer purchasing behavior and generating actionable business insights from retail data.
This project demonstrates a complete analytics workflow using **Python, SQL, and Power BI**, covering data cleaning, analysis, and interactive dashboard development.

---

## 🎯 Project Objective

The objective of this project is to transform raw customer data into meaningful insights that can help businesses:

* Understand customer behavior and preferences
* Identify high-value customer segments
* Analyze sales trends and revenue drivers
* Support data-driven decision making

---

## 🛠️ Tech Stack

* **Python** → Data Cleaning & Exploratory Data Analysis (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL** → Data querying and business analysis
* **Power BI** → Dashboard creation and visualization

---

## 📌 Project Workflow

### 🔹 Data Cleaning & Preprocessing (Python)

* Handled missing values and inconsistencies
* Removed duplicate records
* Prepared data for analysis

### 🔹 Exploratory Data Analysis (EDA)

* Analyzed customer demographics
* Identified purchasing patterns
* Visualized trends using graphs

### 🔹 SQL Analysis

* Performed customer segmentation
* Analyzed revenue and sales performance
* Extracted business insights using queries

### 🔹 Dashboard Development (Power BI)

* Built an interactive dashboard
* Used KPIs, filters, and charts
* Presented insights in a visual format

---

## 📈 Key Insights

* Observed trends in purchasing behavior

## ⚙️ How to Run the Project

### ▶️ Python

1. Open the notebook file
2. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Run all cells

### ▶️ SQL

1. Import dataset into your database
2. Run queries from the `.sql` file

### ▶️ Power BI

1. Open the `.pbix` file in Power BI Desktop
2. Explore the dashboard
